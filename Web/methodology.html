Methodology & Data Documentation

Semiconductor Supply Chain Network · Sahasrik Rudraveera Ragani · Ahmedabad University · 2026

Overview

This project maps the global semiconductor supply chain as a six-layer directed network and applies graph-theoretic metrics to identify structural chokepoints — nodes through which a disproportionate share of global supply-chain traffic must pass.

The core claim is that conventional trade statistics systematically misattribute geopolitical leverage in this industry because they measure bilateral flows between sovereign states, while the actual structural power is concentrated in a small number of private firms whose positions in the network make them obligatory passage points for the entire global technology stack.

Everything in this project is reproducible from public data. The full pipeline is open-source on GitHub and can be re-run from scratch in approximately 30 minutes.

Reproducibility

The R script (01_network_pipeline.R), raw data files, and processed JSON outputs are all publicly available. See Replication Instructions below.

Value-Added Weighting

Trade volume is a poor measure of structural importance because it double-counts: a component that crosses a border three times appears three times in gross trade statistics. We use OECD Trade in Value Added (TiVA) 2025 edition to weight edges by domestic value added in gross exports (indicator: EXGR_DVA), sector C26 (manufacture of computer, electronic and optical products), year 2020.

This means an edge's final weight blends two signals: (1) its biproportional trade share, and (2) the value genuinely created by the exporting country in that sector. A country that assembles chips imported from elsewhere gets lower weight than one that designs or fabricates them.

The R Pipeline

The data pipeline is written entirely in R and is fully reproducible. Every decision point is commented in the script. Below is a plain-English walkthrough of each step.

01

Geocoding Lookup

Natural Earth Admin-0 GeoJSON provides ISO3 country codes and pre-computed label placement coordinates (LABEL_X, LABEL_Y) for every sovereign territory. These become the node longitude/latitude on the map.

02

TiVA Value-Added Scores

OECD TiVA C26 DVA data is loaded and filtered to year 2020 (most complete year in the download). Aggregate codes (OECD, EU27, NAFTA, etc.) are excluded. Values are in USD millions and normalised 0–1 against the maximum (China, $412B).

03

Comtrade Edge Construction

Each Comtrade file contains reporter-to-World aggregates — every country's total exports and imports to/from the world, but not bilateral pairs. We reconstruct bilateral flows using biproportional inference (see Edge Construction below).

04

Passive Consumer Filter

Nodes with in-degree ≥ 2 and out-degree = 0 across all layers are classified as passive consumers — they import chips but play no structural role in the supply chain. These are pruned to reduce noise. See Degree Filter.

05

Betweenness Centrality

Computed per layer using igraph's betweenness() with weights = 1/edge_weight (stronger dependency = shorter path = higher centrality). Normalised 0–1.

06

JSON Export

The pipeline writes three files: nodes.json (39 nodes with all metrics), edges.json (485 edges with layer tags and weights), and meta.json (layer definitions and disclaimer). These are the only files the frontend consumes.

Mathematical Framework I: Betweenness Centrality

What it measures — plain English

Imagine every possible pair of countries in the network wants to send something to each other — raw materials, equipment, chip designs, finished circuits. Each pair finds the most efficient route through the network (the "shortest path"). Betweenness centrality simply counts: for how many of these routes is node i a necessary stop?

A node with high betweenness is like a major highway interchange. Close it and traffic across the entire system backs up. That is why we call it the Toll-Booth Parameter: the higher the betweenness, the more rent a node can extract from the system, and the more catastrophic its disruption would be.

Formal definition

Equation 1 — Betweenness Centrality
SymbolMeaning in this study
\(C_B(i)\)Betweenness centrality of node \(i\) (e.g. TSMC, Netherlands, China). Ranges 0–1 after normalisation.
\(\sigma_{st}\)Total number of shortest weighted paths from source node \(s\) (e.g. a Japan wafer supplier) to target node \(t\) (e.g. a Vietnam assembly plant).
\(\sigma_{st}(i)\)The number of those paths that route through node \(i\). If every path from \(s\) to \(t\) must pass through \(i\), then \(\sigma_{st}(i)/\sigma_{st} = 1\).

How we compute it

We use igraph's implementation of Brandes' algorithm (2001), which runs in O(VE) time. Edge weights are inverted before computation: a strong dependency (weight = 0.98) becomes a short path (distance = 1/0.98 ≈ 1.02), so the algorithm correctly identifies strong-dependency paths as the shortest routes through the network.

g <- graph_from_data_frame(edges %>% select(from, to, weight=weight_norm))
bc <- betweenness(g,
                  directed   = TRUE,
                  weights    = 1 / E(g)$weight,   # invert: strength → shortness
                  normalized = TRUE)               # divide by (n-1)(n-2) for comparability
Key Finding

In four of six supply chain layers, the highest-betweenness actor is a corporation, not a nation-state. ASML (BC = 0.98 in Equipment) and TSMC (BC = 0.95 in Fabrication) are the two dominant chokepoints in the entire network.

Mathematical Framework II: Network Formation Games

Why the network looks the way it does — plain English

Why does every major chip designer — Nvidia, AMD, Qualcomm — send their fabrication work to one company in Taiwan, rather than building their own fabs? The answer is not cultural or historical: it is mathematical.

Every actor in the semiconductor network is trying to maximise a utility function: get the maximum benefit from being connected to other nodes, while spending as little as possible on the direct links. Building a leading-edge semiconductor fab costs approximately $20 billion and takes 5–7 years. Signing a manufacturing contract with TSMC costs a fraction of that. The math decisively favours the contract.

Formal definition

Equation 2 — Network Formation Utility (Jackson & Wolinsky 1996)
SymbolMeaning in this study
\(U_i(g)\)Total utility (profit + supply security) for actor \(i\) in network \(g\). Every state and every firm is trying to maximise this simultaneously.
\(f(d_{ij})\)Benefit of accessing node \(j\). This benefit decays as \(d_{ij}\) increases — having a direct fab contract (distance 1) is more valuable than routing through two intermediaries (distance 2 or 3).
\(c_{ij}\)Cost of maintaining a direct link to \(j\). For fabrication, this is ~$20 billion to build a fab, or a fraction of that for a supply contract. The gap between these two costs is why the star equilibrium forms.

Mathematical Framework III: Pairwise Stability & The Star Equilibrium

Why decoupling is mathematically hard — plain English

A network is pairwise stable when: (1) no existing link is costly enough that one party wants to cut it, and (2) no two unconnected parties both want to add a new link. This is the equilibrium concept for strategic network formation.

When the cost of building a direct link \(c_{ij}\) is sufficiently high — specifically, when \(c_{ij} > f(1) - f(2)\), the benefit of a direct link over an indirect one — the equilibrium network topology is a star: one hub connected to all other nodes, with no direct spoke-to-spoke links.

This is empirically what we observe. TSMC is the hub. The US, China, Japan, South Korea, and Europe are the spokes. No spoke independently builds a direct link to another spoke (i.e. no country builds its own leading-edge fab to supply other countries) because the math says it is cheaper to route through the hub.

The only way to escape a star equilibrium is for a state actor to artificially subsidise \(c_{ij}\) — to reduce the cost of a direct link until the math changes. This is, precisely, what the US CHIPS Act ($52B), the EU Chips Act (€43B), and Japan's semiconductor investment programme ($6.8B) are doing. They are mathematical interventions, not just industrial policy.

Policy Implication

Total decoupling from the TSMC star is not individually rational for any actor under current cost structures. The star equilibrium breaks only when domestic fab costs fall below the threshold \(c_{ij} < f(1) - f(2)\). State subsidies are the mechanism — not trade policy, not diplomatic pressure.

Edge Construction: Biproportional Inference

Comtrade provides reporter-to-World aggregates: each country reports its total exports and total imports to/from the world, but does not always provide bilateral (country-to-country) breakdowns in the format we need.

To reconstruct bilateral edges from these marginal totals, we use biproportional inference (also called the RAS method or gravity model), which is standard practice in international economics for this exact problem:

Equation 3 — Biproportional Edge Weight

Where \(s_i^X\) is country \(i\)'s share of world exports in that HS code, and \(s_j^M\) is country \(j\)'s share of world imports. The product \(s_i^X \cdot s_j^M\) is then normalised so the maximum edge weight = 1.

This means: a country that exports 40% of world semiconductor equipment (Netherlands/ASML) connecting to a country that imports 30% of world semiconductor equipment (Taiwan) gets a weight of 0.12 before normalisation — and after normalisation, relative to the strongest edge in the layer.

exporters <- df %>% filter(flow=="X") %>%
  group_by(iso3) %>% summarise(val=sum(value)) %>%
  mutate(share = val/sum(val))

importers <- df %>% filter(flow=="M") %>%
  group_by(iso3) %>% summarise(val=sum(value)) %>%
  mutate(share = val/sum(val))

edges <- cross_join(
  exporters %>% select(from=iso3, e=share),
  importers %>% select(to=iso3,   i=share)
) %>%
  filter(from != to) %>%
  mutate(weight_norm = (e * i) / max(e * i))
Limitation

Biproportional weights assume that each exporter supplies all importers in proportion to their import shares. This is an approximation. It will overweight edges between large exporters and large importers even when no direct relationship exists. Bilateral Comtrade data, where available, would be more precise.

Degree Centrality Filter: Passive Consumer Pruning

Not every country in the world is a structural participant in the semiconductor supply chain. Many countries import chips inside consumer electronics (laptops, phones) without playing any role in production, design, or distribution. Including these nodes adds noise without adding analytical signal.

We apply a simple rule: any node with in-degree ≥ 2 and out-degree = 0 across all layers combined is classified as a passive consumer and removed from the network.

passive <- degree_tbl %>%
  filter(in_deg >= 2, out_deg == 0) %>%
  pull(iso3)
# Removed: TWN, POL, AUS, ESP, BRA, HUN
The Taiwan Case

Taiwan (TWN) is removed as a passive consumer at the country level. This is not an error — it reflects the structure of Comtrade data. TSMC's output registers as corporate revenue, not sovereign HS 8542 exports. Taiwan's structural importance is correctly captured by the TSMC firm node (BC = 0.95 in Fabrication). We document this explicitly as the "Taiwan Paradox" in our findings.

Limitations

We document these explicitly so the analysis can be properly evaluated and extended.

1. Reporter-to-World Comtrade data
Our Comtrade files contain each country's exports/imports to the World, not bilateral pairs. Biproportional inference approximates bilateral flows. This is a known limitation: it will produce non-zero edges between pairs with no direct relationship.
2. Firm-level betweenness is a proxy
TSMC, ASML, and other firm nodes have betweenness scores derived from market-share literature rather than computed from bilateral trade data. These are well-documented figures from reputable sources, but they are not computed from the same pipeline as country-level scores.
3. Single time period
Trade data is from 2022, TiVA from 2020. The network is a snapshot. It does not capture post-2022 restructuring driven by US export controls, the CHIPS Act, or China's domestic semiconductor investment programme.
4. Taiwan pruning
Taiwan's country node is removed by the passive consumer filter, which is methodologically correct given Comtrade data structure, but requires explicit disclosure. TSMC as a firm node compensates but is not fully equivalent.
5. Layer 2 is structural, not empirical
Refined Materials edges are based on published industry literature rather than a single authoritative data source. The HS codes that cover photoresists, silicon wafers, and specialty gases are scattered across multiple product categories.

Glossary

Betweenness Centrality C_B(i)
The fraction of all shortest weighted paths in a network that pass through node i. Used here as a measure of structural chokepoint power.
Biproportional Inference (RAS)
A method for estimating bilateral flows from marginal totals. Standard in international trade economics for reconstructing trade matrices.
DVA (Domestic Value Added)
The portion of a country's gross exports that represents value actually created domestically, after subtracting imported inputs. From OECD TiVA.
Ego-network
The subgraph consisting of a selected node and all edges directly connecting to it. When you click a node on the map, the visualisation isolates its ego-network.
HHI (Herfindahl-Hirschman Index)
Sum of squared market shares. Ranges 0–1; values above 0.25 indicate high concentration. Used here on betweenness shares per layer.
HS Code
Harmonized System commodity code, the international standard for classifying traded goods. HS 8542 = integrated circuits; HS 8486 = semiconductor manufacturing equipment.
OSAT
Outsourced Semiconductor Assembly and Test. The packaging and testing stage, dominated by Taiwanese firms (ASE Group) and Malaysian facilities.
Pairwise Stability
A network equilibrium concept: no actor wants to delete an existing link, and no two actors mutually want to add a new one.
Passive Consumer
A node with in-degree ≥ 2 and out-degree = 0. Imports goods but plays no structural supply role. Pruned from the network to reduce noise.
Star Equilibrium
A network topology where one hub node connects to all others, with no direct spoke-to-spoke links. Emerges when direct link costs exceed the marginal benefit of bypassing the hub.
TiVA
Trade in Value Added — OECD database measuring the domestic value content of gross exports, correcting for double-counting in standard trade statistics.
